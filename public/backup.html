Backup & Restore

Create Backup

Click the button below to download a complete backup of the current database. This file contains all contests, candidates, scores, and users. Store it in a safe place.

Restore from Backup

To restore the system, you must first unzip your downloaded backup file. Then, select the `tabulation.db` file from within that unzipped folder.

WARNING: This is a destructive action!

Restoring will completely erase and overwrite the current database. This does NOT restore images, only the contest data and scores. Any changes made since the backup was created will be lost. This action cannot be undone.
